# **App Name**: Procurement Portal

## Core Features:

- Secure User Access: User authentication and role-based access control for different customer zones.
- Private Customer Zones: Creation of dedicated, private zones for each customer.
- Opportunity Creation: Creation and management of opportunities within each customer zone.
- Document Checklist Generation: Ability to create dynamic lists of required documents for each opportunity, with clear descriptions and upload areas.
- Document Upload: Document upload functionality within each opportunity to facilitate document collection.
- Alerting Feature: Notifications to alert clients about expiring deadlines.
- Intelligent Document Suggestion: AI tool suggesting missing required documents based on customer profile and opportunity metadata. LLM must reason when or if to provide suggestions, as not all suggestions are needed in every scenario.

## Style Guidelines:

- Primary color: Dark blue (#34495E) to convey professionalism and trust.
- Background color: Light gray (#ECF0F1) for a clean, uncluttered backdrop.
- Accent color: Teal (#2ECC71) to highlight key actions and provide a modern touch.
- Body font: 'PT Sans', a humanist sans-serif suitable for body text.
- Headline font: 'Playfair', a modern serif, suitable for headlines; pair with 'PT Sans' for body
- Use clear and professional icons to represent document types and actions.
- Prioritize a clear, intuitive layout to ensure easy navigation.
- Use subtle transitions to enhance the user experience and provide visual feedback.
